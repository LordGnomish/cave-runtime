# cave-embed

High-throughput embedding service for Cave Runtime — an OpenAI
`/v1/embeddings`-compatible API with a sentence-transformers / BGE / E5 /
Mistral / nomic / jina model registry, dynamic token-budget batching, the
standard pooling strategies, embedding quantization, a result cache, and a
cross-encoder reranker.

Upstream reference: [michaelfeil/infinity](https://github.com/michaelfeil/infinity) (MIT).
cave-embed is AGPL-3.0-or-later.
